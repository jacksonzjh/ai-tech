## 技术栈

```plaintext
JH AI Learning
├── 前端
│   ├── 页面开发：Vue
│   ├── AI大模型集成：moonshot-v1-8k api
│   └── 服务托管：Vercel
│
└── 后端
    ├── AI提示工程：结构化Prompt
    ├── RAG知识库：Langchain
    └── 向量数据库：Qdrant
```
